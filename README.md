<h2>📌 Project Overview</h2>
<p>
The Blog Website is a web application developed to allow users to read and manage blog posts in an organized and user-friendly interface. 
The platform focuses on clean design, smooth navigation, and responsive layout to ensure a good user experience across devices.
</p>

<p>
This project demonstrates practical implementation of front-end web development technologies including HTML, CSS, and JavaScript, along with modern hosting platforms.
</p>

<hr>

<h2>✨ Features</h2>
<ul>
<li>Responsive blog layout</li>
<li>Clean and modern user interface</li>
<li>Blog post display system</li>
<li>Easy navigation between pages</li>
<li>Mobile-friendly design</li>
<li>Fast deployment using Vercel</li>
</ul>

<hr>

<h2>🛠️ Technologies Used</h2>
<ul>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
<li>Firebase (for backend services)</li>
<li>Vercel (for deployment)</li>
</ul>

<hr>

<h2>🚀 Deployment</h2>
